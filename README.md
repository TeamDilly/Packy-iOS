## About
### 패키(Packy), 마음으로 채우는 특별한 선물박스 앱 🎁

기프티콘으로 가득한 세상, 패키는 더욱 특별한 온라인 선물문화를 만들어요. 

온라인 상에서 선물이 아니었던 것도, 패키와 함께라면 선물이 돼요!
